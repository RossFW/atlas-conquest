## Data Files

All files live in `site/data/` and are generated by the pipeline.

| File | Description | Status |
|------|-------------|--------|
| `metadata.json` | Last updated timestamp, data version | Planned |
| `cards.json` | Card definitions (name, faction, type, stats, keywords) | Planned |
| `commanders.json` | Commander definitions and attributes | Planned |
| `matches.json` | Match history (players, decks, commanders, winner, duration) | Planned |
| `card_stats.json` | Aggregated card analytics (play rate, winrate) | Planned |
| `commander_stats.json` | Commander winrates, matchup matrix | Planned |
| `deck_archetypes.json` | Deck clustering and meta breakdown | Planned |
| `trends.json` | Time-series data for meta shifts | Planned |
